# Base de Conhecimento — Imagemweb

Esta pasta contém toda a base de conhecimento acumulada no desenvolvimento dos sites Imagemweb.
Mova-a para qualquer novo projeto e inicie uma sessão Claude com ela como contexto.

## Arquivos

| Arquivo | Conteúdo |
|---|---|
| `stack.md` | Stack técnica, decisões e convenções do projeto |
| `coolify-deploy.md` | Guia completo de deploy no Coolify (API, banco, app, DNS) |
| `seguranca.md` | Práticas de segurança implementadas (auth, upload, env vars) |
| `padroes-api.md` | Padrões de rotas API Next.js (auth, erros, tipagem) |
| `auth.md` | Implementação completa de autenticação JWT + bcrypt |
| `docker-volumes.md` | Docker, volumes persistentes, start.sh, seed |
| `design.md` | Convenções de design (Tailwind v4, cores, componentes) |
| `checklist-novo-projeto.md` | Passo a passo para subir um novo projeto do zero |

## Uso em nova sessão

Ao iniciar um novo projeto, diga ao Claude:
> "Leia os arquivos em `_base-docs/` — eles contêm todo o contexto técnico do projeto anterior."
